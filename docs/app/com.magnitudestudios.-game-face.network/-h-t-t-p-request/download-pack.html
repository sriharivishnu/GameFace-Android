downloadPack

suspend fun downloadPack(context: Context, packID: String, packType: String): Resource<LocalPackInfo?>

Download pack from the server. Downloads the image file, the text content and creates a local folder and stores the results from the download in it. The user should now have these packs online or offline.

Parameters

context - Context (preferably applicationContext)

packID - ID of the pack to be downloaded to the device

packType - The type of the pack to be downloaded

Return
A data class (LocalPackInfo) if successful
